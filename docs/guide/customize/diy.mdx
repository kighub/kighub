## 建模

建模是头壳制作的第一步，很大程度上决定了最终成品的整体形状和细节。

### 3D建模软件推荐
- **Blender**：老牌开源免费软件，功能强大，应用最广泛、教程最多，适合初学者和专业人士。
- **ZBrush**：专业级数字雕刻软件，适合精细细节处理
- **Maya**：工业标准3D建模软件，功能全面但学习曲线较陡
- **3DS Max**：适合硬表面建模，有丰富的插件生态

   - 从球体或立方体开始，逐步调整形状

## 头壳制造

- **3D打印**：快速成型，适合复杂形状。如今消费级3D打印机采购价格相对较低，适合DIY制作。

### 制造工艺

3. **后处理**：去除支撑结构，打磨表面
4. **拼接组装**：如果模型较大，需要分件打印后拼接

### 打磨和上色
1. **表面处理**
   - 粗打磨：使用粗砂纸（80-120目）打磨表面
   - 细打磨：使用细砂纸（240-400目）精细打磨
   - 抛光：使用抛光膏或砂纸（600-1000目）抛光

2. **上色步骤**
   - 底漆：使用白色或灰色底漆，增强颜料附着力
   - 基础色：喷涂基础颜色
   - 细节色：添加细节和阴影
   - 高光/消光漆：添加高光/消光漆，增强质感
   - 保护层：使用清漆或保护喷雾，防止颜色脱落

### 内部结构
1. **内衬**：添加舒适的内衬材料，如海绵、布料
2. **固定带**：安装头带或下巴带，确保头壳稳固

4. **视野调整**：安装机箱防尘网+相纸打印的瞳孔作为眼睛，能在一定程度上增强视野和透气性
