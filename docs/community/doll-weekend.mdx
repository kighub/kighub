# Doll Weekend

[Doll Weekend](https://dollweekend.com/)（简称 DW）是起源于中国、聚焦 Kigurumi（二次元偶装 / 着ぐるみ）亚文化的大型线下集会，2013 年首次举办，核心是让爱好者通过佩戴精致头壳、身着定制衣装进行角色扮演，实现对动漫 / 游戏角色或原创形象的高度还原，同时也是该领域全球规模领先的交流平台。

## 核心信息
- 本质定位：由深圳市未偶时空科技有限公司主办，最初是爱好者自发组织的无偿活动，后来发展为连接 B 端动漫企业与 C 端消费者的行业交流平台，也是全球最大的 Kigurumi 主题展会之一。
- 核心形式：区别于传统 Cosplay，Kigurumi 以覆盖全身的偶装和标志性头套为核心，强调头壳工艺与整体造型的协调，让参与者隐匿真实身份，完全融入角色，呈现出独特的二次元 “娃娃感” 与神秘感。
- 核心理念：不问参与者的国籍、年龄、性别，仅以 “外壳”（头壳与造型）相互认识，为爱好者提供纯粹的展示与交流空间。

## 发展历程
- 2013 年，首次举办，最初是小型爱好者聚会。
- 2024 年 11 月，DW11 在广东举办，吸引众多国内玩家参与。
- 2025 年 8 月，DW12 在广东河源举办，被称为 “有史以来规模最大的偶装动漫展会”，现场有 SkyDoll 无人机表演、偶装才艺秀等活动。
- 2026 年 1 月 1-3 日，DW13 在杭州钱塘君澜大饭店举办，以 “钱塘潮涌，‘偶’遇江南” 为主题，进一步扩大了活动影响力。

## 活动内容与特色
1. **核心活动**
  - 偶装展示与巡游：参与者扮演热门 IP 角色或原创形象，进行集体巡游，成为现场亮点。
  - 才能装才艺表演：设置偶装才艺秀舞台，爱好者展示舞蹈、互动等才艺。
  - 交流互动：为全球爱好者提供线下见面、经验分享的机会，促进圈子文化传播。
2. **配套活动**：常设置无人机表演、烟花秀、主题市集等，部分场次还有限定派对、特色空间打卡等环节，增强沉浸式体验。
3. **独特氛围**：活动多在酒店等封闭且舒适的场地举办，参与者可在较长时间内自由活动，尽情享受角色扮演的乐趣，形成了包容、纯粹的亚文化氛围。
